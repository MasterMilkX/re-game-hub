This is a research experiment to test and compare the creative game design capabilities of large language models (LLMS.) We create 3 base game designed around design principles of 3 genres of games: space shooter, platformers, and rogue-like dungeon crawlers.

For this experiment we prompt the ChatGPT to suggest and provide code snippets for features to improve these base games. We directly implement these suggested features and then compare the games to see which LLM is better at suggesting features that improve the game. We compare the LLM made games against a user-only improved game and the original base game.

This is a user study intended to evaluate the cohesiveness, uniqueness, and game feel of the resulting LLM co-designed games against pure human-made games. The results of this study will be used to inform the design of future LLMs and to help game designers understand the potential and limitations of LLMs in the game design process.

You can read the paper for this experiment here. This was accepted to the 2024 Foundation on Digital Games conference.
